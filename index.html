scarves for your favorite things


A simple, user-friendly code to display favorite books, movies, TV shows, etc on blogs and personal websites, as seen in the live example below.

Downloads
CSS HTML Instructions ▾

STEP 1: Copy the css code (aka "scarfcode1") and paste it above </style> in your site's theme code.

STEP 2: Copy the html code (aka "scarfcode2") and paste it below <body> in your site's theme code.

STEP 3: The default number of scarves is 4. If you want fewer than 4 just remove however many scarves you want in scarfcode2. If you want more than 4, add code identical to scarf4 in scarfcode1 and scarfcode2, name it scarf5, and adjust the margin.

STEP 4: Scarves are just image files. In scarfcode2, replace http://url1, http://url2, and so on with the urls of the fandom scarves you want. You can find the ones I created if you scroll down.

STEP 5: Adjust the position of the scarves for your theme based on the size of your content and if it's center, left, or right aligned. This is done in scarves{...} in scarfcode1.
  • Center-aligned: Increase the value of margin-left to move the scarves further to the right from center. Or, change margin-left to margin-right and increase the value to move the scarves left from center.
  • Left-aligned: Remove "left:50%;" and "right:50%;". Increase the value of margin-left to move the scarves further to the right from the left edge of the page.
  • Right-aligned: Remove "left:50%;" and "right:50%;". Change "margin-left" to "margin-right". Increase the value of margin-right to move the scarves further left from the right edge of the page.
